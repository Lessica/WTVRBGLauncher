# WTVRBGLauncher

中文输入法语音转文字免跳转插件。  
无设置选项，安装并注销后立即生效。

原理：动画移除术  
支持的输入法：微信、百度、搜狗、讯飞

Tested on iOS 15.4.1 / 16.4.1, Dopamine 2.
